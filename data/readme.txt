These are the data extracted from Lindauer et al., 2017 (https://pubmed.ncbi.nlm.nih.gov/27863176/)
Fig2a, Fig2b, Fig3a, FigS2, FigS3. 
Figures are digitized using Web Plot Digitizer (https://apps.automeris.io/wpd/)

mrgsolve.csv is simulation data using mrgsolve. The goal of this data is to compare mrgsolve simulation result with Julia simulation result. See ../Julia_implementation/runRtest.r for more information. 

SobolData.rds is generated by global sensitivity analysis script. The goal of saving this data is to make reproducing global sensitivity analysis figure easier.
